OverviewApp is now easier to use in WordPress, you can easily and seamlessly integrate our plugin that handles all the nasty work for you.

== Description ==

You can easily have your blog working with OverviewApp. This plugin attaches the tracker for you, and attaches it to every footer in your WordPress blog.

== Installation ==

1. Copy the "overviewapp.php" file into your "plugins" directory and enable it
2. Under "Settings" go to "OverviewApp"
3. Enter your API Key and click "Save Changes"

== Frequently Asked Questions ==

= What hook does this rely on? =

It rely's on wp_footer, which all themes use.
